Тема: Каскадні таблиці стилів

CSS дозволяє відокремити оформлення веб-сторінки від її структури. Це значно спрощує підтримку коду та дозволяє швидко змінювати дизайн усього сайту.

* CSS — Cascade Style Sheets (Каскадні таблиці стилів).

Тема: Селектори класів

Селектор класу вибирає елементи з певним атрибутом class. Для вибору елементів з певним класом слід написати символ крапки (.), за яким слідує ім'я класу.

* Класи є чудовим інструментом для багаторазового коду.
